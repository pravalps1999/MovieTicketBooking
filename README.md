# Movie Ticket Booking System

A Spring Boot application designed to manage movie theater operations, including seat bookings and user authentication.

## 🚀 Features
* **User Management:** Secure login and registration.
* **Booking Logic:** Real-time seat selection and reservation.
* **Admin Dashboard:** Tools to add movies and showtimes.

## 🛠 Tech Stack
* **Java** (Spring Boot)
* **Spring Security** (for Authentication)
* **SQL** (Database management)
* **Maven** (Build tool)

## 📖 How to Run
1. Clone the repo: `ggit clone https://github.com/pravalps1999/MovieTicketBooking.git`
2. Start Redis
3. Strat Apache Kafka
2. Run `./mvnw spring-boot:run`
